Conceptos de HTML Básico

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje que se utiliza para crear la estructura de páginas web. Está compuesto por elementos llamados etiquetas.

Etiquetas más comunes

<h1> a <h6>

Se usan para títulos. Por ejemplo:

Este es un ejemplo de título

<p>

Define un párrafo de texto. Por ejemplo:

Este es un párrafo de ejemplo en HTML.

<img>

Muestra imágenes:

<a>

Crea enlaces:

Ir a Google

Ejemplo de código HTML

<!DOCTYPE html>
<html>
<head>
    <title>Mi primera página</title>
</head>
<body>
    <h1>Hola mundo</h1>
    <p>Este es mi primer sitio web.</p>
</body>
</html>
